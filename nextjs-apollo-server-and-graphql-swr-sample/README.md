## SetUp Env Variables

create ".env.local" include below content

```bash
NEXT_PUBLIC_GITHUB_PERSONAL_ACCESSTOKEN=ghp_xxx
```

## Install Library

```bash
$ yarn install
```

## Lint And Foramt

```bash
$ yarn lint:css

$ yarn lint:js

$ yarn format
```

## Serve Dev

```bash
$ yarn dev
```

## Serve Build

```bash
$ yarn build
```

## Serve Prod

- https://nextjs.org/docs/deployment

```bash
$ yarn serve
```
